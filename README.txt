# Документация к коду функции по нахождению площади квадрата 

## Описание

Данный код реализует функцию для вычисления площади квадрата, который был получен с помощью двух симметричных прямых((0;X), где X - любая точка на прямой OX) относительно прямой y=x  
на декартовой системе координат, а также содержит набор тестов для проверки корректности работы этой функции с использованием модуля unittest.

## Функция

### area_square_on_the_coordinate_system(a)

- **Описание**: 
  Функция принимает длину стороны квадрата и возвращает его площадь. Площадь квадрата вычисляется как квадрат длины стороны.

- **Параметры**:
  - a (float или int): Длина стороны квадрата.

- **Возвращает**:
  - float или int: Площадь квадрата.

## Класс тестов

### SquareTestCase

Класс, который содержит тесты для функции area_square_on_the_coordinate_system.

#### Методы:

1. **test_area_of_square_with_positive_side**
   - Проверяет корректность вычисления площади квадрата с положительной длиной стороны.
   - Входные данные: 10
   - Ожидаемый результат: 100

2. **test_area_of_square_with_zero_side**
   - Проверяет корректность вычисления площади квадрата с нулевой длиной стороны.
   - Входные данные: 0
   - Ожидаемый результат: 0

3. **test_area_of_square_with_negative_numbers**
   - Проверяет корректность вычисления площади квадрата с отрицательной длиной стороны.
   - Входные данные: -10
   - Ожидаемый результат: 100

## Запуск тестов

Для запуска тестов необходимо выполнить следующую команду в терминале:


bash
python.exe -m unittest square.py


Где square.py — это имя файла, содержащего данный код.
